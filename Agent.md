# Agent Notes

- The landing offering carousel and its controls should appear on mouse enter and disappear on mouse leave.
- The landing carousel control bar should always read "Slide left and right".
- Account input forms should follow the carousel slider visual language: 75px rounded pill rails, soft grey background, 3px inset padding, and circular icon/action wells.
- Create/account dropdowns should follow the marketplace dropdown pattern: use a relative wrapper, an `appearance-none` select with generous right padding, and a separate `ChevronDown` icon so the arrow is never squashed.
- Create listing form fields should match marketplace filter fields: compact `rounded-[18px]` controls, `bg-orbit-panel`, small label spacing, and no focus outline/ring highlight.
- Do not let elements display outside of their set boundaries unless instructed to.
